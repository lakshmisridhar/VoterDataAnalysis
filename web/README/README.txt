==================ITIS 6120 -Applied Database - Project 3==============================================
			
==================Development and Test Environment====================================

Java Jdk: 1.8.0.66
Tomcat: Apache Tomcat Version 8.0.30
NetBeans IDE: 8.0.2
Database Server : MySQL

===================Steps to execeute======================================

* Install the softwares required
* Download the "Voter Data File" from the Mecklenburg County Board of Elections Website: http://charmeck.org/mecklenburg/county/BOE/data/Pages/default.aspx
* Complete Database set up using the sql files provided
* Unzip the VoterDataApp and import the project as "existing web application" in Net beans IDE
* Clean and build the project . Click run to execute the web app.
